Requirements

To build the Ve classes and application you need:
  • A working OpenGL library
  • The GLUT library
  • The OpenCV computer vision library.
  • The Epix XCLIB frame grabber library. (If you plan to use an Epix frame grabber board.

Windows Build Instructions

The build process has been tested with Visual Studio .NET (VS 7.1), and project files are provided for that version of Visual Studio (won't work with earlier versions).

GLUT library

A pre-compiled GLUT library for Windows can be downloaded from http://www.xmission.com/~nate/glut.html.

Extract the glut32.dll library to your system directory (e.g. C:\Windows\system32) or another directory in your search path. The header files must be in a GL\ subdirectory in the compiler's header search path (e.g. [...]\Microsoft Visual Studio .NET 2003\Vc7\include\).

OpenCV library

A Windows installer for OpenCV can be downloaded from the OpenCV homepage at http://sourceforge.net/projects/opencvlibrary/.

Simply run the installer, and set the path to the OpenCV includes in Visual Studio:

Project -> Properties -> C/C++ -> General -> Additional Include Directories
You also need to set the path to the libraries under:
Project -> Properties -> Linker -> General -> Additional Library Directories
