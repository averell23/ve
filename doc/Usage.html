Usage guide for the Ve software

The ve package contains the ve executable, which is an AR application that uses the ve framework.

Command line parameters

The ve program accepts the following command line parameters and options:
  • -source Specifies the image source that should be used. Valid options are "epix", for the use of an EPIX Inc. frame grabber card and "dummy" for a collection of dummy images.
  • -epixconf Specifiy a parameter file that should be loaded upon opening the EPIX frame grabber library. This parameter only takes effect when the "epix" video source is selected.
  • -debug This forces an specific debug level to be used for all logging output, overruling the settings in the property file. Valid settings are: "trace","debug","info","warn","error" and "fatal". This option will only take effect after the command line has been parsed.
  • -help Causes usage information to be printed out.

Configuration files

Currently the only configuration file used by the application is the log4cplus configuration, supposed to be found in ../logger.properties.

Keyboard commands

When running, the application may be controlled by keystroke commands:
  • Global (Video Canvas)
    • i Acquire an offset correction image from a black camera image
  • Status Display
    • space Toggle on/off
  • Offset Correction
    • o Toggle offset correction on/off
  • Brigthness Control (May not work for all video sources)
    • 1 Decrease left eye brightness by 5%
    • 2 Increase left eye brightness by 5%
    • 3 Decrease right eye brightness by 5%
    • 4 Increase right eye brightness by 5%
  • Camera Calibration
    • c Toggle Calibration Overlay on/off
    • b Blank the unused eye
    • v Toggle left/right eye calibration
    • x Take a snapshot (only saved if patter was recognized
    • n Delete current snapshots
    • m Recalibrate camera parameters from snapshots
  • Sensor/Camera Registration
    • r Toggle Registration overlay on/off
    • t Take new registration point
    • e Toggle left/right eye
    • w Re-register the AR system
  • Tracking/Display
    • n Toggle Crosshairs
    • m Toggle Highlight marker
